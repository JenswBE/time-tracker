# Datamodel

```
=> Global calc balance

=> Day

- Date (uniq1, default from record)
- Project (uniq1, default)

- Working hours (default)
- Invoiced
- Expected end (calculated)

=> Timespan

- Start (same day other day = new record)
- End (same day, other day = new record)

=> Project

- Name

=> Location

- Name
- On-site
```
